At each level of the hierarchy, the “owner” of a domain name can either add hosts or add sub-domains.
    msu.edu Domain name granted to msu by .edu
    pilot.msu.edu Host part which is of the msu.edu domain
    gopher.msu.edu Host which is part of the msu.edu domain
    egr.msu.edu Subdomain of msu.edu delegated to the
    MSU College of Engineering
    scully.egr.msu.edu Host which is part of egr.msu.edu domain
    mi.us Subdomain for the geographical state of
    michigan
    k12.mi.us Subdomain for K12 education in Michigan
    holt.k12.mi.us Subdomain for Holt School District
    scnc.holt.k12.mi.us Host which is part of holt.k12.mi.us
    One thing that makes this a “bit” confusing is that you can often address E-Mail to either a host or a domain.
    crs@msu.edu crs@egr.msu.edu crs@scully.egr.msu.edu
    If you cannot connect to a name using telnet it is probably only a domain name.
