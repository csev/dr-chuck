When a large number of users are viewing non-multicast media simultaneously such as a live broadcast, a complete copy of the data is sent across the network for each user (potentially hundreds).
    Using multicast, only one copy of the data is sent into the network and it is only replicated at the last moment. The “MBONE” Routers detect when there is a “viewer” of the material and only sends the data down a network connection if there is a consumer of the information actively watching the “program”.
    For example if 100 people at Michigan State University were to watch a live program at Stanford University, only one copy of the data would be delivered to MSU and it wouild be replicated only on the different Local Area networks at MSU.
