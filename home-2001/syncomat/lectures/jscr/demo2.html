This is an example of how JavaScript can be used interactively. Type a brief message in the box below and then select one of the display options from the pull-down list. Depending on your choice, your text will be displayed in an alert box, a second browser window, or in the status bar at the bottom of the page.

When you have tried all of the display options, go to the menu bar at the top of the browser and look at the source file for this page. In Netscape, select "View" and then "Page Source". In Internet Explorer, select "View" and "Source". This will allow you to view the JavaScript code that created this page. When you are done looking at the source code, click on the "Go To URL" button to return to the lecture.
