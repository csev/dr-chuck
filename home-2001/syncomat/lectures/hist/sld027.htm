One of the great contributions of the World-Wide-Web was the concept called the Uniform Resource Locator or (URL). URLs may initially look like gibberish, but they tell the Web Browser a great deal of information.
    The first part of the URL specifies a high level network protocol which is needed to retrieve the data. Examines include “http”:, “ftp”:, “news:”. The prefix “http:” indicates to use the Hypertext Transport Protocol to retrieve the data.
    The second part of the URL is the host to contact to retrieve the data. Any domain name or numeric IP address can be used in this spot.
    The third part of the URL specifies a document to be retrieved from the server.
    The suffix on the document also determines how the information is to be displayed. Examples include:
  • GIF - Picture
  • JPG - Picture
  • PS - Postscript File
  • HTML - Hypertext Markup Language
  • RA - RealAudio file
  • MPG - MPEG Animation
