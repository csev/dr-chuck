Software Developed:

I developed a tool called Sync-O-Matic 3000 which automates the production of web-lectures using RealNetworks media and HTML. This product is in use at a number of sites around the world and is downloaded several times each week. The best example of its use is the CERN Web lecture repository http://webcast.cern.ch/Projects/WebLectureArchive/.

To address some of the difficulties in using Sync-O-Matic, I developed a tool called ClipBoard-2000 which also produces web-based lectures using Quicktime. ClipBoard-2000 is very easy to use and it allows the instructor to annotate their slides with text and drawing. ClipBoard also allows the instructors to use a document camera to show objects. ClipBoard is early in the development process but has generated significant interest from both commercial and educational institutions. See http://www-personal.umich.edu/~csev/projects/cb2k/ for more details and sample media.

Books and Chapters:

I am the author for the second edition of "High Performance Computing", published by O'Reilly and Associates. The book was published in 1998. It is designed to be used to teach HPC to programmers of many fields ranging from chemistry to computer science. It has been used around the country in a number of undergraduate HPC courses.

Since 1995 I have been the editor of the Standards Column of the IEEE Computer magazine. I have written 15 columns and edited the remaining columns. Selected Columns: January 1999 - POSIX: A Model for the Future of Computing, July 1998 - Standardizing Real-Time Streaming Protocols, March 1998 - IEEE 754: An Interview with William Kahan, September 1997 - OSI Retrospect and Prospect, and February 1997 - Linking Computers and Consumer Electronics.

I wrote a chapter on "Distributed File Systems" in the book The Internet for Everyone: A Guide for Users and Providers written by Richard Wiggins available from McGraw-Hill.

I participated in the development of IEEE Standard POSIX P1003.0: Guide to the POSIX Open System Environment. I was a member of P1003.0 Working Group from January 1990 through July 1995. I was the primary author for the following sections: Introduction, (4.2) Network Services, and (4.11) Application Development Support Services. This document is about to be approved as an international ISO technical report TR-14292. I was also the vice-chair for the POSIX 1003 committee from 1993-1996.
