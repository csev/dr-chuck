This style is the standard style for the Sync-O-Matic program. It makes good use of the Javascript capabilities of Sync-O-Matic. If browsers fail with this style, then you should try the simpler style.

It is intended that you edit this index.htm file and add your own picture, etc - You may also not want to give all of the above options to the user.

The file inst160.gif in this directory is used to show the instructor picture for audio-only versions of the lecture. You can replace this picture for all lectures using the standard style by replacing {Application Directory}\style\sync2k\inst160.htm on the machine that produces these lectures.

It supports a frame-style view or a separate player and browser window. It is the most heavily used style and is rather complex. If you are developing your own style it might be best to start with a simpler style.
