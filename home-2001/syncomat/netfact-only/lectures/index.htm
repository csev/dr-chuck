Internet and Technology

This page contains most of the lectures in a course developed and taught by Dr. Charles Severance from 1997-1999 while he was at Michigan State University.

This course was one of the very earliest full length college courses delivering the entire lecture content with synchronized multimedia viewable over a modem. The course was much of the impetus for developing and improving Sync-O-Matic. The lectures are here in case you find them interesting.

  • What Is the Internet
  • History of the Internet
  • Basic Internet Tools
  • HTML Overview
  • Basic Imaging
  • Design of Web Pages - Guest Lecturer - Sue Davidsen
  • Business Use of the Internet
  • Intellectual Property - Guest Lecturer - Richard Wiggins
  • Personal Security on the Internet - Guest Lecturer - Nick Lewis
  • A TCP/IP Overview
  • Advanced Internet Tools
  • Javascript - Guest Lecturer - Katy Luchini
  • Active Server Pages - Guest Lecturer - Nick Pellegrini
Currently these are delivered using pseudo-streaming, so any random navigation, such as using thumbnails, will be slow, especially over a modem. Once the streaming server is in place, we will put all of the course lectures up at this site.
