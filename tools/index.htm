Educational software for children

Dr. Times

This is a simple tool which can be used to practice multiplication tables and addition tables. It has support for audio and several modes to provide some type of assistance. You can replace the sounds with your own voice. I hope to rewrite the application using VB.NET in a way which allows voice input as a way to answer the questions. I also want to make it smarter to have a sense of where the student has strengths and weaknesses.

Software to support technology-based education

Clipboard-2000

This is a tool which works on the PC and Macintosh to easily produce QuickTime movies which include video, audio, sildes, scribbling, typing, and annotation. The tool can be used to record live lectures or to create material expressly for offline viewing. It also has a "server mode" where it can be coupled wwith Media Cleaner Pro to dynamically produce lectures as a lecture publishing server. There is a user group for technical support at Yahoo Groups. The download site contains many samples of ClipBoard lectures.

Download Now

Sync-O-Matic 2000

This is another tool to create distance lectures using the Real Media technologies. Sync-O-Matic is somewhat more difficult to use than ClipBoard for the lone user, but it has a significant number of features which make it well suited for a production facility making lectures. Probably the best example of a large repository of Sync-O-Matic lectures is the Cern Web Lecture Archive in Geneva Switzerland. A user discussion group is available on Yahoo Groups.

MAKEHTM - A tool for photo Albums

Makehtm is a simple tool which makes web pages for a set of photos. I don't (yet) have very good documentation, but the idea is to put a bunch of images from a digital camera into a directory with some alphabetical naming convention. Then use makehtm to put a little annotation along with each photo using the Edit feature. Then Makehtm will create thumnails and a bunch of .htm files to make the narrated photo album. ARPYOURBASE

This tool is only useful for a small segment of the population. This tool is absolutely necessary when you are using an Apple Airport to act both as a wireless gateway and a home dial-up Internet router and you want to use PC's running Windows-98 and Windows-95 on the Ethernet segments of your network. There is a problem with ARP entries (it is hard to assign the blame between Apple and Microsoft, both work poorly). The solution is to force a refresh of the ARP entry for the gateway every 100 seconds. This little tool does that in a nice convienent way. Source code is included in the download and shows how to do some nifty IP operations from Visual Basic.
