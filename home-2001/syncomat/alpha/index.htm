Downloading Sync-O-Matic 2000

  • Make sure to view the On-Line documentation about installation. You must install Realplayer 7 and RealProducer 7 before installing Sync-O-Matic.

  • Get your free registration key for the product by joining the on-line mailing list to eliminate the nagging splash screen.

  • Pick your release:
    • Version Beta 5 - (01/07/01) What is new

    • Version Beta 4 - (12/31/00) What is new

    • Version Beta 3 - (07/12/00) What is new - Debug Version - Download - (07/21/00) Documentation
    • Version Beta 2 - (06/21/00) What is new
    • Version Beta 1 - (06/18/00) What is new
    • Alpha Test Page (old)

  • Follow the instructions in the on-line documentation

This will replace any existing Sync-O-Matic installation, so to use Sync-O-Matic 3000 (older version) you must reinstall it over top of Sync-O-Matic 2000 Alpha. It is no big deal to install one and then the other. Just try to make sure to uninstall one before reinstalling the other.

Source Code

The source code to Sync-O-Matic is available for purchase for sites who would like to have a copy for their own use. If your institution has a significant repository of Sync-O-Matic materials which can be referenced in the gallery of web-based lectures, please contact us regarding special purchase terms.

The source code to some of the more complex modules with interactions with various ActiveX controls is available in hopes that other folks can help figure out good cannonical ways to interact with these ActiveX controls. Part of the goal is for people to test and examine these controls and send back feedback/bugs so these capabilities can be improved in future versions of Sync-O-Matic.
