This slide shows two of the functions that can be called inside of the “Act()” function that was described in the previous slide.
    The function “showMessageBox()” is defined at the top of this slide. It takes the user’s text and displays it in an alert box on the screen.
    The function “showWindow()” is also defined on this slide. It creates a new browser window using the window.open() function, then writes the user’s text to that new window.
