Inside the <BODY> of the HTML document, the <SCRIPT> tag is used again to call the function displayMessage(), which was defined previously in the <HEAD>. When the Internet browser interprets this part of the JavaScript code, it will execute the instructions for displayMessage() which are defined in the <HEAD> of the document.
    Note that HTML comment tags <!-- --> are used to hide the JavaScript code from older versions of Web browsers that cannot interpret these scripts. This is sometimes called “cloaking” a JavaScript. In an older version of the browser, this code is simply ignored.
